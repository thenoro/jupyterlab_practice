## Dependencies
The following packages are needed to run the Jupyter Notebooks: 
  - python=3.12
  - conda==24.5.0
  - setuptools
  - pip
  - wheel
  - conda-content-trust
  - conda-libmamba-solver
  - menuinst
  - console_shortcut_miniconda
  - powershell_shortcut_miniconda
  - anaconda-anon-usage
  - jupyter-book
  - jupyterlab
  - jupyterlab-git
  - jupyterlab-spellchecker
  - pandas
  - nb_conda_kernels
